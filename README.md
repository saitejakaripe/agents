# Agents

A hands-on repository for building and experimenting with multi-agent AI systems using notebooks, memory workflows, and orchestration frameworks such as LangGraph, AutoGen, and MCP.

## Overview

This project explores practical agent engineering through step-by-step notebooks and supporting data resources. It covers foundational concepts, LLM integrations, graph-based orchestration, multi-agent workflows, and memory-enabled tooling.

## Topics Covered

- Agent foundations
- OpenAI-based workflows
- LangGraph orchestration
- AutoGen-based multi-agent systems
- MCP-based tool integration
- Memory and persistence workflows
